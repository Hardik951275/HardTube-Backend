# HardTube Backend

YouTube API Proxy Backend built with FastAPI + Python for Flutter App.

## Features
- YouTube Search & Trending
- Video Details
- User Authentication (JWT)
- Watch History
- Real-time ready

## Setup

1. Clone the repo
2. Get YouTube API Key from Google Cloud
3. Copy `.env.example` to `.env` and fill values
4. Run with Docker:
   ```bash
   docker-compose up --build
